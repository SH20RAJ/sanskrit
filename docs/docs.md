# Sanskrit Documentation

## Language Guide

### Basic Syntax

Sanskrit uses Devanagari script for keywords and identifiers. Here are some basic syntax examples:

#### Variables
```sanskrit
चर x = 5;
चर नाम = "राम";
```

#### Functions
```sanskrit
कार्य योग(क, ख) {
    वापस क + ख;
}
```

#### Control Flow
```sanskrit
यदि (x > 0) {
    प्रिन्ट("धन");
} अन्यथा {
    प्रिन्ट("ऋण");
}
```

### Data Types

- संख्या (Number)
- पाठ (String)
- बूल (Boolean)
- सूची (Array)
- वस्तु (Object)
